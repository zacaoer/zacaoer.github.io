教程

git下载

git clone https://github.com/abishekvashok/cmatrix

编译

./configure
make
make install

使用方式

  1. 常用参数
    parameter explanation
    -b 随机粗体
    -u 刷新频率，0-9
    -c 显示的颜色，支持green,red,blue,white,yellow,cyan,magenta and black
  2. 示例
    更改屏保为随机粗体，红色字母：
    cmatrix -b -C red
    更改屏保为异步滚动：
    cmatrix -a
